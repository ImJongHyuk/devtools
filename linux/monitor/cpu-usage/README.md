# cpu-usage

## 코어별 CPU 사용량 측정
- `measure_cpu_usage.sh`
- 현재는 1초에 한번 측정

## 측정한 사용량 시각화
- `cpu_plot.py`
- 측정 결과 파일을 소폭 수정 필요함.

측정된 파일 구조
```json
{},
{},
...
{},
```

변경할 파일 구조는 `"data"`로 묶어줘야함
```json
{
    "data": [
        /** 측정 파일 내용 **/
        {},
        {},
        ...
        {}
        /** 마지막 , 제거 **/
    ]
}
```
